## 1. 概述

## 3. SSH 远程连接配置

### 3.1 本地 SSH 配置文件

文件路径：`C:\Users\Administrator\.ssh\config`

```
Host 2080
    HostName 100.102.182.117
    User xfgzs
    Port 22
    IdentityFile ~/.ssh/id_ed25519

Host yongjia
    HostName 100.83.28.28
    User root
    Port 22
    IdentityFile ~/.ssh/id_ed25519
```

### 3.2 密钥文件

| 文件 | 路径 |
| --- | --- |
| 私钥（本地） | `C:\Users\Administrator\.ssh\id_ed25519` |
| 公钥（本地） | `C:\Users\Administrator\.ssh\id_ed25519.pub` |
| 公钥（2080远程） | `~/.ssh/authorized_keys`（xfgzs 用户目录下） |

认证方式：Ed25519 密钥认证（免密登录）

### 3.3 快速连接

```bash
# 连接 2080 服务器（Ubuntu）
ssh 2080

# 连接 yongjia 服务器
ssh yongjia
```
