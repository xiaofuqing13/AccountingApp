# 粉色少女主题更新说明

## 调整目标
- 将全局视觉从蓝系切换为粉色少女风。
- 保持原有布局结构和交互，仅替换主题色与状态色，避免功能回归。

## 调整范围
- 明亮主题色板：`app/src/main/res/values/colors.xml`

- 主色 `pink_primary`、辅助色 `pink_light/pink_soft`、渐变 `gradient_*` 全部切换为粉紫系。

- 文本提示色、边框色、波纹色同步改色，保证页面一致性。
