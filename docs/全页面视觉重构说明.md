# 全页面视觉重构说明

## 目标
- 解决“整个页面设计不统一、观感混乱”的问题。
- 统一首页、记账、日记、会议、设置为同一套少女风视觉语言。

## 本次重构范围
- 页面头部统一：
  - 新增 `app/src/main/res/drawable/bg_page_header.xml`
  - 记账页、日记页、会议页统一为圆角渐变头部
- 列表页统一：
  - `app/src/main/res/layout/fragment_account_list.xml`
  - `app/src/main/res/layout/fragment_diary_list.xml`
  - `app/src/main/res/layout/fragment_meeting_list.xml`
  - 统一搜索栏、空状态、FAB 风格与间距
- 列表项卡片统一：
  - `app/src/main/res/layout/item_account.xml`
  - `app/src/main/res/layout/item_diary.xml`
  - `app/src/main/res/layout/item_meeting.xml`
  - 全部改为描边卡片、弱阴影、同圆角体系
- 文案与交互修复：
  - `app/src/main/java/com/loveapp/accountbook/ui/account/AccountListFragment.kt`
  - `app/src/main/java/com/loveapp/accountbook/ui/meeting/MeetingListFragment.kt`
  - `app/src/main/java/com/loveapp/accountbook/ui/adapter/AccountAdapter.kt`
  - `app/src/main/java/com/loveapp/accountbook/ui/adapter/MeetingAdapter.kt`
  - `app/src/main/java/com/loveapp/accountbook/ui/adapter/DiaryAdapter.kt`
  - 修复乱码提示、统一金额符号、统一对话框文案

## 验证
- 构建：`gradlew.bat :app:assembleDebug` 通过
- 真机烟测：安装后切换底部五个页面，未出现 `FATAL EXCEPTION`
