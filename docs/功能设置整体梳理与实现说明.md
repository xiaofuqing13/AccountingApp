# 功能设置整体梳理与实现说明

## 这次的处理思路
- 不只改外观，按“可配置、可保存、可扩展”重做设置页。
- 先把高频设置做成真功能，再保留你现有数据导入导出能力。

## 已落地功能
- 数据与备份
  - 导出备份（`xlsx`）
  - 导入备份（智能识别）
  - 导入旧版备份
  - 导入示例数据（带确认）
  - 存储路径展示与一键复制
  - 自动备份开关（本地持久化）
- 通用
  - 深色模式开关（即时生效 + 持久化）
  - 消息提醒开关（本地持久化）
  - 隐私锁入口（保留原彩蛋逻辑）
- 关于
  - 版本号动态显示
  - 意见反馈入口
  - 情话小屋入口
  - 版本连点彩蛋保留

## 代码改动
- `app/src/main/res/layout/fragment_settings.xml`：设置页结构和样式重构
- `app/src/main/java/com/loveapp/accountbook/ui/settings/SettingsFragment.kt`：交互逻辑重构
- `app/src/main/java/com/loveapp/accountbook/util/AppSettings.kt`：统一设置持久化

## 下一步建议（可继续做）
- 自动备份从“开关”升级为“定时策略”（每天/每周）。
- 消息提醒接入系统通知通道（可设提醒时段）。
- 隐私锁升级为 PIN/生物识别（不再仅彩蛋入口）。
