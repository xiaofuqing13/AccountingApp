# 日记左滑编辑删除改造（2026-02-20）

## 改造目标
- 将日记列表从“卡片常驻编辑/删除按钮”切换为“左滑触发编辑/删除”。

## 交互规则
- 日记列表项向左滑动：在卡片右侧露出“修改 / 删除”按钮。
- 点击卡片：查看详情。
- 长按卡片：不再作为编辑/删除入口，统一改为左滑触发。

## 代码变更
- `app/src/main/java/com/loveapp/accountbook/ui/diary/DiaryListFragment.kt`
  - 使用 `ItemTouchHelper` 实现左滑并限制滑动距离。
  - 左滑后固定露出按钮区，不再弹出二级操作弹窗。
  - 增加开合动画、右滑收起（轻拉即可回收）与动态阈值，滑动手感更平滑。
- `app/src/main/java/com/loveapp/accountbook/ui/adapter/DiaryAdapter.kt`
  - 新增 `onEditClick/onDeleteClick` 回调，绑定左滑露出的按钮点击事件。
  - 增加展开项状态管理，确保同一时间仅一条卡片展开。
- `app/src/main/res/layout/item_diary.xml`
  - 新增卡片底层操作区（`btn_swipe_edit`、`btn_swipe_delete`），前景卡片左滑露出按钮；操作区样式简化，避免圆角边缘不顺滑。

## 验证
- 编译通过：`:app:compileDebugKotlin`
- 安装通过：`:app:installDebug`
- UI 树验证：日记列表中不再存在 `btn_edit/btn_delete` 常驻控件。
