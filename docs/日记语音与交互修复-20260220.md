# 日记语音与交互修复说明（2026-02-20）

## 本次修复内容
- 日记详情弹窗新增 `删除` 按钮，进入详情后可直接 `编辑/删除`，不依赖左滑或长按。
- 日记列表卡片保留并强化 `编辑/删除` 直接按钮交互（按钮始终可见）。
- 语音标记解析增强，兼容以下历史数据格式：
  - 大小写混用（如 `[audio:xxx]`）
  - 冒号前后含空格（如 `[AUDIO : xxx]`）
  - 旧数据中带路径、URL 编码、`file://` 前缀的文件名
- 语音文件定位增强：
  - 优先兼容旧绝对路径
  - 自动规范化为文件名再回落到 `diary_audio` 目录
- 语音播放增强：
  - 播放前统一走兼容数据源配置（路径 + `FileDescriptor` 双通道）
  - 增加 `AudioAttributes`，提升设备兼容性
  - 播放失败时给出明确提示（不再“点击无反应”）
- 语音文件名生成策略优化为毫秒级时间戳，降低同秒录音覆盖/误删风险。
